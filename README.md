# Party Spin Wheel

A lightweight React and TypeScript spin-wheel game built with Vite. Players can spin a colourful wheel to select a random challenge, reward, or mini-game.

## Getting started

```bash
npm install
npm run dev
```

## Commands

- `npm run dev` starts the local development server.
- `npm run build` type-checks and creates a production build.
- `npm run lint` checks the source with ESLint.
- `npm run preview` previews the production build.

## Branches

- `main` is the stable baseline.
- `develop` is the active integration branch for new work.

See [docs/APP_PLAN.md](docs/APP_PLAN.md) for scope, component structure, and milestones.
